**import** *module_name*
``` python
var = module_name.thing_in_module
```


**from** *module_name* **import**  *thing_in_module*
``` python
var = thing_in_module
```

Aliasing import: 
imports a thing as a alias
**from** *module_name* **import**  *thing_in_module* **as** *TiM* 
``` python
var = TiM
```

**from** *module_name* **import** _*_   
``` python
var = thing_in_module
```
