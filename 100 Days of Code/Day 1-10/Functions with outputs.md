# Function with outputs:
Function can return an output. When the function is called it will take the value of the output.
## Python syntax
``` python
def my_function():
	results = 3 + 2
	return results
print(my_function()) #prints: 5
#or save the output:
the_number_five = my_function()
print(the_number_five) #prints: 5
```
